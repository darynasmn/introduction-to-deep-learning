# Introduction to Deep Learning — Exercises

My solutions to the exercises from the **Introduction to Deep Learning (I2DL)** course.

The exercises follow the lectures and gradually build up from the basics of machine
learning to modern deep learning architectures. Along the way I implemented the core
building blocks mostly from scratch and then used them to train and evaluate models.

## What this covers

- **ML & DL foundations** — the general setup: data, models, training loops, and the
  submission workflow.
- **Neural networks** — building fully-connected networks and understanding how they
  learn.
- **Optimization & backpropagation** — computing gradients and updating parameters,
  plus techniques for scaling and stabilizing optimization.
- **Training neural networks** — practical aspects of getting models to train well
  (initialization, regularization, tuning).
- **Loss functions & activations** — the components that shape what and how a network
  learns.
- **Convolutional neural networks** — CNNs for image data and common CNN architectures.
- **Sequence models** — networks for sequential/temporal data.

## Structure

Each `exercise_XX` folder is a self-contained assignment with its own code and notebooks,
corresponding to one topic from the course.
